# sh3-packing-tools

# CREDITS GOES TO BELEK666 @belek666

Silent Hill 3 arc files packer + unpacker by @belek666


To unpack drop *.arc file onto 'arc_unp.exe', file 'arc.arc' need to be in the same folder as archive. Files will be unpacked to 'data' folder. To pack files back drop the same *.arc file onto 'arc_pck.exe'. If a file exist in 'data' folder then it will be placed in archive other ways original file from archive. The new *.arc_new file will be created.
